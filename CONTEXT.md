# Context

## Language

**Chrome** — The installed `/Applications/Google Chrome.app`, using Browser Kit profile `agent` for agent-driven work.
_Avoid_: Chromium-family browser, Chrome-compatible browser

**Brave** — The installed personal Brave browser; eligible for control only when explicitly requested.
_Avoid_: Chrome, default browser

**Chrome for Testing** — An isolated managed test runtime, never a substitute for real Chrome proof.
_Avoid_: Chrome, real-user browser

**Real-environment proof** — Evidence gathered in real Google Chrome with the dedicated `agent` profile.
_Avoid_: browser test, Playwright proof
