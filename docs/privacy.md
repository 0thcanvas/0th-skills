# Privacy Policy

0th Skills is a local workflow plugin for Claude Code and Codex. The plugin does not include analytics, telemetry, or a remote service operated by 0th Canvas.

The plugin can read and write files in the workspace when the host agent is allowed to do so. Any data sent to Claude, Codex, GitHub, or other tools is governed by the user's configuration and those providers' policies.

The plugin's secret-handling contract is documented in `README.md`: agents should handle secret names and references, not resolved secret values.
